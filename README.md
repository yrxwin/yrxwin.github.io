## 技术博客
#### local installation
`git clone --recursive <repo-name>` or 

`git clone <repo-name>` then `git submodule update --recursive --init`

`npm install`

#### start local server:
`npm start`

#### create new blog:
`hexo new "<blog name>"`

#### deploy (after CI setup, we do not need to run this):
`npm run deploy`
